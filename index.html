じぶん書店 森博嗣入門

講談社が「じぶん書店」というサービスを開始しました。
https://www.jibunshoten.com/bookstore/dgfmZCKn

上記のように、自分の好きな本を、好きなように並べた「自分だけの書店」を作成できます。
思う存分に特集を組んで布教ができる、本好きのためにあるようなサービスですね。
試し読みやアフィリエイトもあるようです。

さっそく森博嗣氏の特集を作成。
https://www.jibunshoten.com/bookstore/dgfmZCKn/groupBooks/871

講談社縛りなのが悩ましいですが、面白いサービスなので積極的に使っていきたいと思います。

Hello World

ブログ作りました

kusyuaです。

  • twitter とかQiita(ほとんど使ってない)以外にもoutputの場を設けたい
  • GitHub Pages で遊びたい

この辺の理由から、ブログを作ってみました。
想定しているコンテンツは以下の通り。

  • 作ったものを公開
  • ゲームの感想/攻略/メモ
  • 読んだ小説の感想/レビュー
  • その他

現状何のビジョンもありませんが、ぼちぼち増やせていけたらいいなー
記事、レイアウト、他サービスとの連携等そのうちやっていきます。多分。

ブログ環境について

ブログ環境は以下を使用。

  • GitHub Pages
  • Hexo(静的サイトジェネレータ)

GitHub Pages

GitHub PagesはGitHubのリポジトリをそのままWebサイトとして公開できるサービス。
サーバサイドは弄れないものの、天下のGitHub様にサーバ管理を丸投げできるという素晴らしいサービス。
しかも無料。遊び場にはもってこいです。ありがたや……
https://pages.github.com/

Hexo

静的サイトジェネレータをちゃんと比較した訳ではありませんが、
以下の点からHexoを導入。

  • ブログに特化
  • Markdownが使える
  • javascript
  • NodeJsで簡単に構築/管理できる

githubにリポジトリを用意してしまえば、すべてNodejsで作れるあたり、面倒臭がりにも安心です。

npm install -g hexo
hexo init 'ブログ名'
これだけでだいたいの環境が揃います。多分これが一番早いと思います。
https://hexo.io/
